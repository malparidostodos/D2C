README - ProDetailing (Refined: Jeton · Sofi · Lusion)
------------------------------------------------------------
Contenido del ZIP:
- index.html
- style.css
- script.js
- README.txt

Características principales:
- Diseño inspirado en jeton.com, sofihealth.com y lusion.co
- Tipografía refinada (Space Grotesk + Inter)
- Hero full-bleed con tipografía espaciada y CTAs prominentes
- Sección de servicios con secuencia pin + scrub controlada por scroll (GSAP + ScrollTrigger)
- Paquetes en sección full-bleed, galería y before/after refinado
- Microinteracciones en botones y tarjetas

Instrucciones rápidas:
1. Descomprime el zip en una carpeta.
2. Abre index.html en el navegador.
3. Las animaciones usan GSAP/ScrollTrigger vía CDN — requiere conexión a internet.
4. Reemplaza placeholders por tus imágenes reales para un mejor resultado.
5. Si quieres la versión en React + Framer Motion o integración de reservas, te lo hago.

Notas:
- Código modular y comentado. Puedes ajustar variables en :root (paleta, radios, tamaños).
- El diseño es mobile-first; prueba en diferentes anchos y reemplaza imágenes SVG/placeholder.
